# 安全政策 🔐

[English](SECURITY.md) · [简体中文](SECURITY.zh-CN.md)

**最后更新 2026-07-08 · 发布者：Poietic Studio**

> 安全政策以[英文版](SECURITY.md)为准，本页为要点摘译。

Token Forest 是一款闭源、本地优先、默认不联网的桌面应用。我们认真对待任何安全与隐私行为问题——尤其是与已发布[隐私声明](docs/PRIVACY.zh-CN.md)不一致的行为。

## 报告漏洞

请**私下**报告疑似漏洞，不要为未修复的安全问题开公开 issue：

1. **首选**：本仓库 *Security* 标签页 → *Report a vulnerability*（GitHub 私密漏洞报告）。
2. 邮件：[contact@tokenforest.com.au](mailto:contact@tokenforest.com.au)——主题以 `[SECURITY]` 开头。（专用邮箱 `security@tokenforest.com.au` 将在首个公开版本前启用。）

报告中请**不要**附带：你的 Claude/Codex 日志、提示词或对话内容、源代码、access/refresh 令牌、他人的排行榜数据。如需材料，我们会安排最小化的私密传输方式。

## 重点关注

- 任何本地日志、提示词、对话内容或源代码文件被上传；
- **排行榜关闭时出现任何网络请求**（应用承诺为零）；
- 同意弹窗/隐私声明与实际发送内容不一致；
- 排行榜认证或行级安全绕过（读取/修改他人记录）；
- 令牌泄露、跨系统用户读取本地数据、任意代码执行、下载/签名被篡改、删除流程失效。

请只使用自己的账户与测试数据，问题一经证实即停止。

## 响应目标

目标而非保证：3 个工作日内确认收到，7 个工作日内初步评估，确认的问题至少每 14 天同步进展。严重问题可能立即下架受影响的下载。修复会从干净提交重新构建、重新签名、发布新校验值，并在 Release notes 中公告；被撤下的版本会保留标记而非悄悄改写历史。

## 隐私请求

删除排行榜记录、了解数据处理等请求不属于漏洞——见[隐私声明](docs/PRIVACY.zh-CN.md)的联系方式。**请勿**把 `account.json` 里的令牌发给任何人。

## 下载真实性

官方下载只来自[官网](https://www.tokenforest.com.au)与本仓库 Releases，均附 SHA-256 校验值与签名状态说明。校验不一致的文件不要运行——删除后从官方渠道重新下载，仍不一致请报告。
