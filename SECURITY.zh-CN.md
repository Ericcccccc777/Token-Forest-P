# 安全政策 🔐

[English](SECURITY.md) · [简体中文](SECURITY.zh-CN.md)

Token Forest 是一个本地优先、默认不联网的桌面应用，但我们仍会认真对待任何安全问题。

## 报告漏洞

请**不要**为安全问题开公开 issue。请改为发邮件至：

**[contact@tokenforest.com.au](mailto:contact@tokenforest.com.au)** —— 邮件主题以 `[SECURITY]` 开头

请附上你发现的问题、复现步骤，以及应用版本 / 操作系统。我们会力争在 **72 小时**内确认收到，并在调查和修复过程中持续向你同步进展。

## 范围

- Token Forest 桌面应用（Windows / macOS）
- Token Forest 官网与排行榜服务
