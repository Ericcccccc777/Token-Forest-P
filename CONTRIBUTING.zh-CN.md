# 参与贡献 🌿

[English](CONTRIBUTING.md) · [简体中文](CONTRIBUTING.zh-CN.md)

感谢你想帮 Token Forest 一起成长！应用的源代码在私有仓库里，所以这不是一个"发 PR"式的项目——但你在这里能做的事不少，而且每一件都真切地影响我们接下来做什么：

## 🐛 报告 Bug

发现哪里坏了？[提交 bug 报告](../../issues/new/choose)。操作系统、应用版本，以及你用的 AI 工具（Claude Code / Codex）能让报告可被快速处理——模板会一步步引导你。

## 💡 提出功能建议

[路线图](docs/ROADMAP.zh-CN.md)朝着光的方向生长。[功能请求](../../issues/new/choose)就是你打光的方式。告诉我们你想解决的问题，而不只是解决方案——这能帮我们设计出真正对的东西。

## 🌍 帮忙翻译

应用支持 English 和 中文；官网另有 日本語 和 한국어。发现别扭或错误的措辞？开个 issue，附上原文和你的建议——母语者的反馈是金子。

## 📸 晒出你的森林

大树、奇怪的 bug、温馨的桌面布置——欢迎在 issue 里（以及你喜欢的任何地方——记得 @ 官网！）晒截图。看到真实的森林在野外生长，是这个团队最好的燃料。

## 📝 修正文档

本仓库文档里的错别字或不清楚的措辞？欢迎直接对这里的 markdown 文件发 PR。

## 我们无法接受的

- 对应用代码的贡献（源码是私有的）
- 复用美术素材——它们保留一切权利，见 [LICENSE.md](LICENSE.zh-CN.md)

---

在 issue 和讨论中请保持友善——见我们的[行为准则](CODE_OF_CONDUCT.zh-CN.md)。🌳
