# Contributing 🌿

[English](CONTRIBUTING.md) · [简体中文](CONTRIBUTING.zh-CN.md)

Thanks for wanting to help Token Forest grow! The app's source code lives in a private repository, so this isn't a "send a PR" kind of project — but there's plenty you can do here, and all of it genuinely shapes what we build:

## 🐛 Report bugs

Found something broken? [Open a bug report](../../issues/new/choose). OS, app version, and which AI tool you use (Claude Code / Codex) make a report actionable — the template will walk you through it.

## 💡 Propose features

The [roadmap](docs/ROADMAP.md) grows in the direction of the light. [Feature requests](../../issues/new/choose) are how you point the light. Tell us the problem you're trying to solve, not just the solution — it helps us design the right thing.

## 🌍 Help with translations

The app ships in English and 中文; the website adds 日本語 and 한국어. Spotted awkward or wrong wording? Open an issue with the text and your suggestion — native-speaker feedback is gold.

## 📸 Show us your forest

Screenshots of big trees, odd bugs, and cozy setups are all welcome in issues (and wherever you like to post — tag the website!). Seeing real forests in the wild is the best fuel this team gets.

## 📝 Fix our docs

Typos or unclear wording in this repository's documents? PRs to the markdown files here are welcome.

## What we can't accept

- Code contributions to the app (the source is private)
- Reuse of the artwork — it's all-rights-reserved, see [LICENSE.md](LICENSE.md)

---

Be kind in issues and discussions — see our [Code of Conduct](CODE_OF_CONDUCT.md). 🌳
