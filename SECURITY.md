# Security Policy 🔐

[English](SECURITY.md) · [简体中文](SECURITY.zh-CN.md)

Token Forest is a local-first desktop app with no network access by default, but we treat any security concern seriously.

## Reporting a vulnerability

Please **do not** open a public issue for security problems. Instead, email:

**[contact@tokenforest.com.au](mailto:contact@tokenforest.com.au)** — subject line starting with `[SECURITY]`

Include what you found, steps to reproduce, and the app version / OS. We aim to acknowledge reports within **72 hours** and will keep you updated as we investigate and fix.

## Scope

- The Token Forest desktop app (Windows / macOS)
- The Token Forest website and leaderboard service

Thank you for helping keep the forest safe. 🌳
