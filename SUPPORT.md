# Support 💬

[English](SUPPORT.md) · [简体中文](SUPPORT.zh-CN.md)

Need help with Token Forest? Pick the door that fits:

| I want to… | Go here |
| --- | --- |
| Report a bug | [Open a bug report](../../issues/new/choose) |
| Suggest a feature | [Open a feature request](../../issues/new/choose) |
| Ask a question | Check the [FAQ](docs/FAQ.md) first, then [email us](mailto:contact@tokenforest.com.au) |
| Report a security issue | See [SECURITY.md](SECURITY.md) — please don't use public issues |
| Download the app | [tokenforest.com.au](https://www.tokenforest.com.au) |

When reporting a bug, the issue template will ask for your OS, app version, and which AI tool you use (Claude Code / Codex) — filling those in gets you a fix much faster.

We're a small team; we read everything, usually within a couple of days. 🌱
