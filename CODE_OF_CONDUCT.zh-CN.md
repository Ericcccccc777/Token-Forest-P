# 行为准则 🌳

[English](CODE_OF_CONDUCT.md) · [简体中文](CODE_OF_CONDUCT.zh-CN.md)

Token Forest 社区是一座花园：当每个人都用心呵护时，它长得最好。

## 我们的标准

**请做：** 友善而有耐心，默认善意，给出建设性反馈，尊重不同的观点与经验水平，讨论时不跑题。

**请勿：** 骚扰、侮辱或贬低他人；公开他人的私人信息；发垃圾信息或钓鱼引战；或以任何会让正常人想要离开的方式破坏这个空间。

本准则适用于所有项目空间——issue、讨论，以及任何官方社区渠道——以及在公开场合代表本项目时。

## 执行

违规行为可举报至 **[contact@tokenforest.com.au](mailto:contact@tokenforest.com.au)**。举报由维护者审阅，保密处理，并可能视情节导致警告、删除评论或封禁，由维护者酌情决定。

本行为准则的精神取自 [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) v2.1。
